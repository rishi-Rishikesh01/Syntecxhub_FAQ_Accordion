## ✨ Features
- Smooth open/close accordion animation
- Search bar — filter questions in real time
- Category filter — Internship, Projects, Submission, Technical
- Dark mode toggle with glassmorphism effect
- Minimal clean light mode
- Fully mobile responsive

- CSS3 (Glassmorphism, Animations, Flexbox)
- Vanilla JavaScript
